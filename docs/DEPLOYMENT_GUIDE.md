# CS One CRM Deployment Guide - Phase 11

## Requirements
- PHP 8.1+
- MySQL 8+
- Web server pointing document root to `public/`
- Cron access for `cron/worker.php`

## Setup
1. Copy `.env.example` values into your real config strategy.
2. Create `config/database.php` from `config/database.example.php` if you use the example file.
3. Run `install.php` on a fresh install.
4. For upgrades, run:
   ```bash
   php scripts/migrate.php
   ```
5. Verify health:
   ```
   /public/health.php
   ```

## Phase 11 operational checks
- Set `stripe_webhook_require_verification=1` before production webhooks.
- Configure `stripe_webhook_tolerance_seconds` for your environment.
- Set `login_rate_limit` and `login_rate_window_minutes` to match your security policy.
- Review API analytics and export a baseline CSV before launch.
- Test the billing checkout preview endpoint generated by the app before wiring live Stripe SDK calls.

## Cron
Run the worker every minute or five minutes depending on volume:
```bash
php cron/worker.php
```


## Phase 12 Deployment Notes

- Set `api_rate_limit_per_minute` in System Settings before exposing the API externally.
- Review password policy settings before first user import.
- Use the Diagnostics page after install to verify runtime readiness.
- Review Webhook Events after test deliveries to confirm verification and processing status.
